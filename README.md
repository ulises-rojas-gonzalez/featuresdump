Football League Management es una aplicación diseñada para administrar una liga de fútbol. Permite registrar equipos y entrenadores, almacenar los resultados de los partidos y consultar la tabla de posiciones de forma sencilla.

El proyecto busca facilitar la organización de torneos mediante una estructura clara y fácil de mantener. Además, fue desarrollado siguiendo la metodología Scrum, utilizando Product Backlog, Sprint Backlog, Historias de Usuario, SRS y un diagrama Entidad-Relación para documentar los requisitos del sistema.
